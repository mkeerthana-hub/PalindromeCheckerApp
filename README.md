# PalindromeCheckerApp

## Objective
To check whether a given string is palindrome or not.

## Project Flow
1. Create GitHub repository
2. Clone into local system
3. Create new branch for use case
4. Implement palindrome logic
5. Push to GitHub
6. Create Pull Request and merge

## Use Case 1
User enters a string.
Program reverses the string.
Program compares original and reversed string.
Displays whether it is palindrome or not.
